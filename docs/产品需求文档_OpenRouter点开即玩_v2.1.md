# 产品需求文档：OpenRouter 点开即玩 v2.1

## 1. 目标
访客进入网站后，按“配置页 -> 对局页 -> 复盘页”的固定流程完成一局游戏。

核心目标：
- 无注册、无登录
- 可为每个 AI 单独选择 OpenRouter 模型
- 对局中可看到日志与 Agent 输出
- 对局结束自动进入复盘页

## 2. 用户流程
1. 进入配置页（`/`）
2. 设置房间参数与玩家信息
3. 点击“开始对局”
4. 跳转对局页（`/game/{game_id}`）
5. 进行人类操作或 AI 自动推进
6. 对局结束自动跳转复盘页（`/replay/{game_id}`）

## 3. 页面需求

## 3.1 配置页
必须包含：
- 最大人数（2-6）
- 房间名称
- 玩家列表
  - 玩家名称
  - 真人/AI
  - AI 调用方式（OpenRouter）
  - AI base URL
  - AI 模型下拉（8-10 个可用模型）
- API Key 暂不在页面录入

## 3.2 对局页
布局固定为：
- 顶部：全局状态
- 左侧主体：地图 + 资产 + 联盟
- 右侧：事件日志 + Agent 输出流
- 底部：用户操作区（掷骰子、动作提交、触发 AI、AI 自动推进）

## 3.3 复盘页
- 回合步进
- 跳转指定回合
- 候选动作与最终动作对比
- 策略标签
- 摘要导出（Markdown/JSON）

- 模型下拉项可正常下发并选择
- 页面布局满足“上状态、下操作、左地图、右日志+输出流”
